# 台股分析工具

一個純本機執行的台股分析儀表板，不需要付費資料源或 API 金鑰。後端是 Python 標準函式庫寫的 HTTP 伺服器（無第三方套件依賴），前端是原生 JavaScript，資料即時向 TWSE / TDCC / Yahoo Finance / TAIFEX 的公開 API 抓取。

核心目的是輔助**判斷明天能買哪些股票**——提供技術面、基本面、籌碼面的整合篩選與視覺化，**不是**價格預測工具，所有數字都只是分析參考，不構成投資建議。

## 需要的環境

- Python 3.9 以上（僅用標準函式庫，不需要 `pip install` 任何東西）
- Windows / macOS / Linux 皆可（`啟動.bat` 僅限 Windows 雙擊使用）

## 如何啟動

**Windows：**
雙擊 `啟動.bat`，會自動開啟終端機視窗啟動伺服器，並開啟瀏覽器連到 `http://localhost:8787/`。

**手動啟動（任何平台）：**
```bash
python server.py
```
然後瀏覽器開啟 `http://localhost:8787/`。

關閉伺服器只要關掉終端機視窗，或按 `Ctrl+C`。

## 功能

- **個股走勢圖**：K 線（台股紅漲綠跌）、5/20/60 日均線、布林通道、成交量副圖，支援日線與當日分鐘走勢（9:00–13:30）
- **明日觀察清單**：整合「選股：月線多頭＋財務體質」高分股與「處置股回檔觀察」名單，一鍵產生每日觀察重點
- **多因子選股篩選器**：綜合月線趨勢、營收年增/月增、EPS、籌碼集中度、大戶持股人數變化、流動性等多項指標算出綜合評分（`fit_score`），並標示籌碼營收背離、疑似出貨等風險警示
- **大戶持股追蹤**：透過集保結算所（TDCC）集保股權分散表，追蹤 1,000 張以上大戶持股週對週變化
- **處置股 / 注意股標記**：整合 TWSE 公告資料，並額外監控處置股回檔至布林中線以下的進場觀察名單
- **產業資金流向**：以成交值加權計算各產業當日漲跌幅
- **市場總覽**：加權指數、櫃買指數、費城半導體指數、台指期（日盤/夜盤）即時報價與歷史走勢
- **連線狀態監控**：本機無法連上證交所等資料源時，自動顯示維護畫面提示使用者

## 資料來源（皆為公開、免金鑰）

| 來源 | 用途 |
|---|---|
| TWSE OpenAPI (`openapi.twse.com.tw`) | 個股日線、月均價、月營收、EPS、處置股/注意股公告 |
| TWSE 即時報價 (`mis.twse.com.tw`) | 個股與加權/櫃買指數即時報價 |
| TDCC 集保結算所開放資料 (`smart.tdcc.com.tw`) | 集保股權分散表（大戶持股） |
| Yahoo Finance 公開圖表 API | 當日分鐘走勢、費半指數、加權/櫃買指數歷史資料 |
| TAIFEX (`mis.taifex.com.tw`) | 台指期即時報價 |

所有資料皆延遲數分鐘、僅供參考，不保證即時性或完整性。

## 已知限制

- 台指期（TXF）沒有免費的歷史 K 線來源，走勢圖以加權指數現貨走勢作為標示過的替代參考
- TDCC 資料每週更新一次，工具需要跨越至少兩次更新週期才能顯示真正的持股變化
- 「籌碼集中度」是以成交量估算的近似指標（OBV 概念），並非真實的券商分點進出資料
- 指數（大盤/櫃買/費半/台指期）不支援分鐘走勢圖，僅個股支援
